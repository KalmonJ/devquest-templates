# Build your own Formatter (DevQuest)

Neste desafio você vai construir o seu próprio Formatador de Código (como o Prettier). Passaremos por Lexing, Parsing sintático e a lógica de Pretty-Printing.

## Estágio 1: O Lexer / Scanner
No estágio 1, sua tarefa é receber um arquivo bruto e transformá-lo em uma lista de Tokens para a formatação.

### Como rodar
```bash
npm install
npm run start test.ts
```

### Submissão
Quando achar que o código funciona:
```bash
devquest submit
```
