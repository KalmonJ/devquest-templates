# Build your own HTTP Framework (DevQuest)

Neste desafio você construirá seu próprio framework web (similar ao Express ou Hono) usando Sockets TCP nativos.

## Estágio 1: Servidor HTTP bruto com TCP
Você deve instanciar um servidor TCP na porta 3000 e responder "Hello, World!" com `200 OK` via cabeçalhos HTTP válidos.

### Como rodar
```bash
npm install
npm run start
```

### Submissão
Quando achar que o código funciona:
```bash
devquest submit
```
