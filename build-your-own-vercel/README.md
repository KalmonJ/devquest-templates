# Build your own Vercel (DevQuest)

Crie a sua própria plataforma Serverless (como Vercel/Netlify) construindo uma CLI, deploy em Child Processes, VHosts Roteamento HTTP, Edge CDN e APIs dinâmicas Serverless.

## Estágio 1: CLI Básica e Comandos Iniciais
Monte a estrutura contendo uma sub-CLI que avalia argumentos como `deploy`, `build`, etc.

### Como rodar
```bash
npm install
npm run start deploy
```

### Submissão
Quando achar que o código funciona:
```bash
devquest submit
```
