# Build your own Shell (DevQuest)

Bem-vindo ao desafio "Construa seu próprio Shell"!
Neste desafio, você criará um shell funcional parecido com o Bash ou zsh, lidando com repl (Read-Eval-Print Loop), builtins (como exit, cd, echo), execução de binários rodando no path, redirectioons e piping!

## Estágio 1: Iniciando o REPL e respondendo ao comando exit
Seu primeiro estágio na plataforma é iniciar o loop REPL com o prompt `$ ` (dólar espaço).
Em seguida, ele deve responder ao comando `exit 0` se encerrando sem erros.

### Como testar localmente
```bash
npm install
npm start
```
Digite `exit 0` no console.

### Submissão
Quando achar que o código funciona:
```bash
devquest submit
```
